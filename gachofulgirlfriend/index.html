How to Play ...

The combat section of this game has two aspects:
  1. The Rhythm Game (right hand side)
    2. A selector will cycle through dialogue options, hit 'enter' to select an option. There will be 1 correct option and 3 incorrect ones. Selecting too many incorrect options will result in a interaction failure.
  2. The Bullet Hell Game (left hand side)
    3. Use WASD or the arrow keys to move your brain and dodge anxieties (bullets). Each 'hit' results in the rhythm game to become faster.
